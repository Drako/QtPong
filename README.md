# QtPong

QtPong is a simple Pong game implemented using modern C++11 and Qt5.
Computer-AI can be modified using QtScript.

## Requirements

* CMake (>= 2.8.11)
* Qt5 (Core, Gui, Widgets, Script)
* A modern C++ compiler
